# Keep XP Mod for Minecraft 1.12.2

## Description
This mod allows players to keep their XP after dying while preventing XP orbs from dropping.

## Features:
- Prevents XP from dropping on death.
- Restores XP immediately after respawn.
- Works on Forge 1.12.2.

## Installation:
1. Download `keepxpmod-1.0.jar` from the [Releases Page](https://github.com/HunterMuratore/KeepXPMod/releases).
2. Place it inside your `.minecraft/mods/` folder.
3. Run Minecraft with Forge 1.12.2.
